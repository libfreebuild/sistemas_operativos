Elementary OS

El sistema operativo elemental es una distribución de Linux basada en Ubuntu.
Es la distribución principal para mostrar el entorno de escritorio Pantheon.
La distribución se promueve a sí misma como un reemplazo "rápido y abierto" a macOS y Windows.
Se enfoca principalmente en usuarios no técnicos, y tiene un modelo de pago que desea.
El sistema operativo es desarrollado por elementary LLC

Las pautas de interfaz humana del proyecto de SO elemental se centran en la facilidad de uso inmediato con una curva de aprendizaje suave, en lugar de una personalización completa.
Las tres reglas básicas que los desarrolladores establecieron fueron "concisión", "configuración accesible" y "documentación mínima".
Desde su inicio, el sistema operativo elemental ha recibido tanto elogios como críticas por su diseño, Wired afirmó que el sistema operativo se parece mucho al de macOS tanto visualmente como en términos de experiencia de usuario.
Sin embargo, los desarrolladores elementales disputan esto.
El shell principal de Pantheon está profundamente integrado con otras aplicaciones de SO elementales, como Plank (un dock), Epiphany (el navegador web predeterminado) y Code (un simple editor de texto). Esta distribución utiliza Gala como administrador de ventanas, que se basa en Mutter.

Desarrollo

La distribución de SO elemental comenzó inicialmente como un conjunto de temas y aplicaciones diseñados para Ubuntu que luego se convirtió en su propia distribución de Linux.
Al estar basado en Ubuntu, es compatible con sus repositorios y paquetes y, antes de la versión 0.4 "Loki", usó el centro de software de Ubuntu para manejar la instalación / eliminación de software.
Sin embargo, después del lanzamiento de Loki, se incluyó el paquete elemental en su propia tienda de aplicaciones, llamado AppCenter.
Su interfaz de usuario pretende ser intuitiva para los nuevos usuarios sin consumir demasiados recursos. El sistema operativo elemental se basa en las versiones de Soporte a largo plazo de Ubuntu, que los desarrolladores de Ubuntu mantienen activamente para detectar errores y seguridad durante años, incluso mientras el desarrollo continúa en la próxima versión.
El fundador de SO de primaria, Daniel Foré, dijo que el proyecto no está diseñado para competir con proyectos de código abierto existentes, sino para ampliar su alcance.
El proyecto también busca crear trabajos de código abierto a través de recompensas para desarrolladores ubicadas en tareas de desarrollo específicas.
A partir del lanzamiento de Loki 2016, se habían recaudado US $ 17,500 en recompensas.
