Windows8

Historia:

Windows7 es un sistema operativo de computadora personal que fue producido por Microsoft como parte de la familia de sistemas operativos Windows NT.
Se lanzó a la fabricación el 22 de julio de 2009 y estuvo disponible en general el 22 de octubre de 2009.
Windows 7 fue pensado principalmente para ser una actualización incremental a Microsoft Windows, con la intención de abordar la mala recepción crítica de Windows Vista a la vez que mantiene la compatibilidad de hardware y software.
Windows 7 continuó las mejoras en Windows Aero (La interfaz de usuario introducida en Windows Vista) con la adición de una barra de tareas rediseñada que permite "anclar" aplicaciones y nuevas funciones de administración de ventanas. Otras funciones nuevas se agregaron al sistema operativo, incluidas las bibliotecas, el nuevo sistema de uso compartido de archivos HomeGroup y el soporte para entrada multitáctil.
También se agregó una nueva interfaz de centro de acción para proporcionar una visión general de la seguridad del sistema y la información de mantenimiento, y se hicieron ajustes al sistema de Control de cuentas de usuario para que sean menos intrusivo.
En contraste con Windows Vista, Windows 7 fue generalmente elogiado por los críticos, quienes consideraron que el sistema operativo era una mejora importante respecto a su predecesor debido a su mayor rendimiento, su interfaz más intuitiva (con elogios dedicados a la nueva barra de tareas), menos usuarios Las ventanas emergentes de Control de Cuenta y otras mejoras realizadas a través de la plataforma.
Windows 7 fue un gran éxito para Microsoft. En solo 6 meses, más de 100 millones de copias se habían vendido en todo el mundo, aumentando a más de 630 millones de licencias hasta julio de 2012. En agosto de 2018, aproximadamente el 39% de las computadoras que ejectuan Windows tienen Windows 7.

Desarrollo:

Originalmente se planificó una versión de Windows con el nombre de código "Blackcomb" como el sucesor de Windows XP y Windows Server 2003 y 2000. Las principales características se planearon para Blackcomb, includo un énfasis en la búsqueda y consulta de datos y un sistema de almacenamiento avanzado llamado WinFS para permitir tales Escenarios.
Para el 2003 se anunció un lanzamiento intermedio, de menor importancia, cuyo nombre en código era "Longhorn", lo que retrasó el desarrollo de Blackcomb.
Cuando se lanzó Windos Vista fue criticado por su largo tiempo de desarrollo, problemas de rendimiento, compatibilidad irregular con el hardware y software existente en el lanzamiento, cambios que afectaban la compatibilidad de ciertos juegos de PC y garantías poco claras por parte de Microsoft de que ciertas computadoras que se envían on XP antes del lanzamiento ser "Vista Capable".
Seis meses después del lanzamiento público de Vista, se informó que la próxima versión de Windows se llamaría Windows7, con planes para su lanzamiento final dentro de tre años.
Bill Gates en una entrevista sugerio que Windows Vista sería más dirigido hacia el usuario.
También dijo que se centraría en mejoras del rendimiento
Más tarde Steven Sinofsky amplió este punto, explicando en el blog de Ingeniería de Windows 7 que la compañia estaba utilizando variedad de nuevas herramientas de seguimiento para medir el rendimiento de muchas áreas de el sistema operativo de forma continua, para ayudar a localizar rutas de código ineficientes y para ayudar a prevenir regresiones de rendimiento.
