Windows98:

Explicación:

Su nombre en clave es Memphis, se publicó el 25 de junio de 1998.
Es el sucesor de Windows 95 y fue publicado por Microsoft
Es un producto monolítico de 16 y 32 bits.
Incluía nuevos controladores de hardware y el sistema de archivos FAT32 que soportaba particiones mayores de 2GB permitidos por Windows 95.
Ésta edición es una de las últimas de PC-9821
Su carácterística más notable era la capacidad de compartir entre varios equipos una conexión a internet a través de una sola línea telefónica y algunas mejoras al Windows 98 original.

Conceptos de Windows 98:

Al ser un sistema operativo se encarga de gestionar todos los procesos que ocurren en el ordenador.
En ésta versión se incorporan todas las novedades aparecidas desde los años 1995 hasta el años 1998.
Su principal diferencia de las versiones anteriores(Windos95,Windows 3.10) es que utiliza el sistema de archivos FAT32, lo que lo hace más rápido y almacena datos más eficazmente
Los programas se utilizan más rápidamente y el equipo utiliza menos recursos del sistema

Nuevas y actualizadas características:

Enter las nuevas características de Windows 98 está un mejor soporte AGP, controladores USB funcionales, y soporte para múltiples monitores y WebTV.
Fue la primera versión de Windows en soportar ACPI.
