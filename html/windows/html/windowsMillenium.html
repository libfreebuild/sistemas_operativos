Windows Millennium

Windows Millenium

Historia:

Es un sistema operativo gráfico desarrollado por Microsoft como parte de su familia de sistemas operativos Windows.
Es el sucesor de Windows 98, se lanzó a la fabricación el 19 de junio de 2000 y alcanzó la disponibilidad general el 14 de septiembre de 2000.
Windows ME fue una continuación del modelo de Windows 9x y aún estaba basado en DOS como sus predecesores, pero con acceso restringido al modo real de MS-DOS para disminuir el tiempo de arranque del sistema.

Desarrollo:

En 1998, Microsoft declaro queno habría una versión Windows 9x después de Windows 98. Sin embargo, en mayo de 1999, Microsoft lanzó Windows 98 Segunda edición y luego anunció la nueva versión de Windows 9x que luego se reveló que se llamaba Milleniu. En 2000, se lanzó como Windows Millenium Edition
Al menos tres versiones de Windows ME estaban disponibles durante su fase de desarrollo.
Agregarón un par de características nuevas, como la protección de los archivos de el sistema y el panel de control de opciones del juego.
La fecha de presentación final de Windows Millenium Edition fue el 14 de septiembre de 2000.
Microsoft finalizó el soporte general para Windows Millenium el 31 de diciembre de 2003 y el soporte extendido finalizó en julio 11, 2006.
El soporte extendido de Windows 98 y Windows 98 SE terminó el mismo día.
Windows ME contenía la máquina virtual de Microsoft de Microsoft, lo que provoco que tanto Windows 98 como Windows 98 SE se retiraran de la red de desarrolladores de Microsoft a fines de 2003.
