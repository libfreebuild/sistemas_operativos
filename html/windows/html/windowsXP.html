Creador de Windows:

Windows XP es un sistema operativo de computadora personal producido por Microsoft parte de la familia de sistemas operativos Windows NT.
Se lanzó a la fabricación el 24 de agosto de 2001 y se lanzó ampliamente para la venta minorista el 25 de octubre de 2001.

El desarrollo de Windows XP comenzó a fines de la década de 1990 como Neptune un sistema operativo contruido en el kernel de Windows NT que fue diseñado especificamente para el uso general del consumidor.
Como tal Windows XP fue la primera edición para el consumidor de Windows que no se basó en MS-DOS.
Tras su lanzamiento Windows XP recibió versiones generalmente positivas, con críticos que notaron un mayor rendimiento y estabilidad
Una interfaz de usuario más intuitiva, soporte de hardware mejorado y capacidades multimedia ampliadas.
Sin embargo, algunos revisores de la industria estaban preocupados por el nuevo modelo de licencias y el sistema de activación del producto.
El soporte extendido para Windows XP finalizó el 8 de abril de 2014, después de lo cual el sistema operativo dejó de recibir soporte adicional o actualizaciones de seguridad para la mayoría de los usuarios.

Desarrollo:

El desarrollo inicial de lo que se convertiría en Windows XP se centró en dos productos individuales, "Odyssey" que supuestamente pretendía tener éxito en el futuro Windows 2000 y "Neptune" que según se informa fue un sistema operativo orientado al consumidor que utiliza la arquitectura de Windows NT, sucediendo al Windows 98 basado en MS-DOS.
Los proyectos demostraron ser demasiado ambiciosos. En Enero de 2000, poco ante del lanzamiento oficial de Windows 2000, el escritor de tecnología informó que Microsoft había archivado a Neptune y Odyssey en favor de un nuevo producto con nombre en código Whistler.
