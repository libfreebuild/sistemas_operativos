Windows 95:

Explicación:

Windows 95 es un sistema híbrido con interfaz gráfico de 16 y 32 bits.
Fue lanzado por la empresa Microsoft en 1995, un 24 de agosto.
Mientras de desarrollaba era conocido como Windows 4

Historia:

En ésta versión de el software lanzado por Microsoft se pudieron ver significativas mejoras respecto a las versiones anteriores, concretamente en la interfaz gráfica de Windows, siendo completamente distinta de las versiones anteriores.
Pasó de ser una arquitectura donde se usaban 16 bits multitarea cooperativa a ser una arquitectura multitarea apropiativa de 32 bits.
La versión Windows 95 fue la pionera en incluir barra de tareas y botón de inicio, lo cual se siguieron incluyendo en las versiones posteriores de Windows.
Fue la primera versión capaz de soportar la función de Plug and Play.
La versión de Windows 95 fue vendido en disquetes y en CD-ROM.
La versión de disquete incluían 13 discos
Se utilizaba un formato estándar conocido como DMF el cual permitía almacenar una mayor capacidad mayor a la normal de 1440KiB.
La versión en CD-ROM ofrecía una selección de accesorios y complementos multimedia, además de algunos controladores de dispositivos, juegos y versiones de demostración de algunos programas ofrecidos por la compañia

Sistemas de archivos y nombres largos:

Al utilizar la arquitectura de 32 bits en el acceso a ficheros, se pudo añadir al sistema el uso de nombres largos de archivos.
En la primera versión de Windows 95 se utilizaba FAT16 como sistema de archivos, era accesible con las versiones anteriores de MS-DOS.
Se incluyó el sistema de archivos FAT32 el cual incluía como novedad el soporte de particiones mayores de 2GiB.

Requisitos mínimos:

Los requisitos oficiales que indicaba Microsoft eran los siguientes

-PC con un procesador 486 DX4 o superior
-4MiB de memoria
-Espacio en el disco duro necesario normalmente para actualizar a Windows 95: 35 a 40MB.
-Espacio en disco duro necesario normalmente para instalar Windows 95 en un sistema limpio: 50 a 55MB
-Una unidad de disco de 3,5 pulgadas de alta densidad
-Resolución VGA o superior

Interfaz gráfica de usuario:

Extrenaba una nueva interfaz gráfica de usuario más sencilla y potente que sus antecesores, convirtiendo el sistema operativo en dos años en el mayor éxito de todos los tiempos.

Salida al mercado:

Windows 95 salió al mercado con una fuerte campaña de marketing, incluyendo un anuncio publicitario con la canción de los Rolling Stones.
La campaña publicitario de Microsoft que costó unos 300 millones de dólares mostraba historias de personas haciendo colas en los exteriores de la tiendas, tan sólo p ara conseguir una copia.
En Nueva York, el rascacielos Empire State se preparó para que se iluminara coincidiendo con los colores del logotipo de Windows.
Windows 95 marca la introducción del botón Inicio y la barra de tareas los cuales aún siguén presentes hasta Windows 10
