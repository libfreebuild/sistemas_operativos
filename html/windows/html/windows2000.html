Windows 2000:

Explicación:

Es un sistema Operativo que fue creado por Microsoft en el año 2000, el sucesor de Windows NT. Windows 2000 fue sucedido por Windows XP
Fue un esfuerzo por unir dos sistemas operativos, Windows 9x y Windows NT.
Windows 2000 era un sistema operativo para empresa y ejecutar servidores de red o los servidores de archivo.
Dentro de las tareas que puede realizar se incluyen: crear cuentas de usuarios, asignar recursos y privilegios, actuar como servidor web, FTP, servidor de impresión, DNS o resolución de nombres de dominio, servidor DHCP, entre otros servicios básicos.
Otra de las funciones que tiene es que puede utilizarlo como otra estación de trabajo en la red.
Este sistema operativo es muy eficiente y su principal punto fuerte es Active Directory, una herramienta desde la cual puede administrar toda la infrastructura de una organización.
En este sistema opertaivo, se introdujeron algunas modificaciones respecto a sus predecesores como el sistema de archivos NTFS 5, con la capacidad de cifrar y comprimir archivos. Introdujo tambien las mejoras en el sistema de componentes COM, introduciendo COM+ que unifico en un solo paquete de los servicios anexados a la tecnología COM y MTS de windos NT4, con nuevas ventajas en el ambito empresarial.

Mejora de ésta versión:

En esta versión se pueden encontrar grandes mejoras respecto a la versión anterior, entre las que se encuentran, abundancia de herramientas de conectividad, madurez de la interfaz, buen reconocimiento del hardware y estabilidad. Se añade a esto el soporte de nuevas tecnologías, las mejoras en sus funciones informática remota, aplicaciones centralizadas de servicio y reinicio obligatorios drásticamente reducidos. Muchas de las mejoras de Windows 2000 son sutiles, pero en conjunto crean una mejor experiencia en el uso de un ordenador.
